Local Weather Station

Home weather is a backyard weather station created for my final year project at Curtin University.

About


The weather station is located about 250m above sea level. Situated in the Perth Hills, Local Weather takes observations every 15 minutes.
Data is stored using Microsoft Azure free credit for Students using a MySQL database.
Code is written in Python as well as HTML, PHP, CSS and SQL.
The station was installed on 8 October 2021 at approximatly 5pm however changes continued to occur which may have disrupted results.

Technical specifications


Processor

Local Weather runs on a Raspberry Pi. My version uses a Raspberry Pi 4 with 4GB RAM. The actual resources are significantly lower so a lower model of Pi can be used.

Temperature, Pressure and Humidity

Temperature, Pressure and Humidity are measured using a BME280 sensor More info

Rain and Wind

The anemometer and rain guage is a sparkfun Weather Meter Kit. Rain is measured in increments of 0.2794mm. More info
